Species Tagger for Adobe Lightroom Classic

This installs the Species Tagger plug-in for the current user. No administrator password is needed, and there is nothing to configure — Lightroom Classic finds the plug-in automatically the next time it starts.

You need Adobe Lightroom Classic and Google Chrome installed to use it.

If you previously installed Species Tagger by hand through the Plug-in Manager, remove that old entry after this install (File ▸ Plug-in Manager ▸ select the old Species Tagger ▸ Remove) so it isn't listed twice.
