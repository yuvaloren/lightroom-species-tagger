Species Tagger is installed ✓

Where to find it: select photos in the Library, then choose
File ▸ Plug-in Extras ▸ Identify and Tag Species

How it works: a Chrome window opens with Google Lens results for your photo. You read them, highlight the species name, and press Tag — the plug-in writes the common and Latin names as keywords.

If Lightroom Classic was open during this install, restart it so it picks the plug-in up. To uninstall later: File ▸ Plug-in Manager ▸ Species Tagger ▸ Uninstall. Quick-start guide: github.com/yuvaloren/lightroom-species-tagger/wiki
